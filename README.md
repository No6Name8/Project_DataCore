# DataCore — AI-Driven SME Lending Engine
Hackathon prototype for Alinma Bank's iz Business app.
Replaces static loan applications with real-time POS, bank, and energy data streams.

## Pipelines
- **SME Engine**: Live POS + energy data → dynamic DSCR → auto credit limit
- **Incubator**: Salary/DBR assessment → seed loan → transitions to SME pipeline

## Phases
- Phase 1: Dashboard UI
- Phase 2: Mock Data Generation + Simulated API
- Phase 3: AI Models (DSCR + DBR)
- Phase 4: Full Integration
